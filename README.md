# AMIRA ALDAHABI GOLD

## Luxury Gold Jewelry E-Commerce Website

A modern React-based e-commerce website for a luxury gold jewelry dealer.

### Features
- Browse luxury gold jewelry products
- Add items to shopping cart
- Real-time cart total calculation
- Elegant black and gold design theme

### Products Available
- Rope Chain - $150
- Cuban Bracelet - $300
- Gold Ring - $180
- Custom Pendant - $220

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Technologies Used
- React 18
- Vite
- JavaScript
